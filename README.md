# nota

This repository currently centers on [`inv/`](./inv/), which contains `nota`, a single-owner invoicing app built with Next.js, Drizzle, Neon, Stripe, and Resend.

## Primary app

- [`inv/`](./inv/): `nota`, a production-focused invoicing app with PDF and XRechnung export, Stripe payment links, and Vercel-ready cron/webhook routes
