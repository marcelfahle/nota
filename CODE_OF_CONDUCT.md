# Code of Conduct

This repository expects professional, respectful collaboration.

## Expected behavior

- Assume good intent, but be direct and specific.
- Focus discussion on code, product behavior, and evidence.
- Share feedback that is actionable and respectful.
- Help keep the repository welcoming to contributors with different backgrounds and experience levels.

## Unacceptable behavior

- Harassment, personal attacks, or discriminatory language
- Deliberate intimidation or trolling
- Publishing private information without permission
- Bad-faith disruption of issues, pull requests, or discussions

## Enforcement

The maintainer may edit, remove, or reject contributions and discussions that violate this code of conduct.
